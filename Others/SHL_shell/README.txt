Bonus are available from the main function (bdsh):
    -where command (example: ./bdsh -f file.json select user firstname,id where lastname=SMITH)
    -update command (example: ./bdsh -f file.json update user firstname=Marry where firstname=John)
    -delete command (example: ./bdsh -f file.json delete user where lastname=SMITH)

Unit test are available from the test folder (test):
    -unit test (example: ./test.bash)
